# Financial Management

## Features

- [ ] User accounts
    - [ ] Creating a new user account
        - [ ] Users have an email
        - [ ] Users have a password
        - [ ] Users can have a display name
        - [ ] Users can have a phone number
    - [ ] Editing existing user account
        - [ ] Users can change their password
        - [ ] Users can change their display name
        - [ ] Users can change their phone number
    - [ ] Logging in to an existing user account
        - [ ] Users can log in with their email and password
- [ ] Transactions
    - [ ] Creating new transactions
        - [x] A transaction is the amount of money that a user either paid or received
        - [ ] A transaction is associated with a user
        - [x] Transactions have a description
        - [x] Transactions have an amount
        - [x] Transactions have a date
    - [ ] Editing existing transactions
        - [ ] Description of a transaction can be changed
        - [ ] The amount of a transaction can be changed
        - [ ] The date of a transaction can be changed
    - [ ] Viewing existing transactions
        - [ ] The list of all transactions can be viewed
        - [ ] The list of all transactions can be searched based on containing a certain string in their description
        - [ ] The list of all transactions can be filtered by date range
        - [ ] The list of all transactions can be filtered by amount range
        - [ ] The list of all transactions can be filtered by tags
- [ ] Debits and credits
    - [ ] A debit is a transaction that is owed by a user
    - [ ] A credit is a transaction that is owed to a user
    - [ ] Debits and credits are created from transactions
    - [ ] Debits and credits have all properties of a transaction
    - [ ] Debits and credits have a party (person or company)
    - [ ] A debit or credit is associated with a user
- [ ] Tags
    - [ ] A transaction can have one or more tags
    - [ ] A tag is a simple piece of text that help categorize transactions
